# Tribute Page: Nelson Mandela

This project is a tribute page created to honor the life and legacy of Nelson Mandela. It is built using **HTML** and **CSS** and demonstrates responsive design, semantic elements, and hover effects.

## Features

- Responsive layout with semantic HTML5 elements.
- Featured image with descriptive caption.
- Biography section with styled text and hover animations.
- Smooth hover animations using `transform` and `box-shadow`.
- Linear-gradient background for a visually appealing design.
